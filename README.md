A simple Blog built with Node.js/Express.js - https://simple-crud-node-blog.herokuapp.com
## Installation
Install all dependencies, edit the .env file.
```sh
npm install
```

## Run
```sh
npm start
```

## Built With
```sh
- Node.Js
- Express.Js
- MongoDB
```
